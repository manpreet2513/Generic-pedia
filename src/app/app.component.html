GenericPedia – Encyclopedia of Generic Drugs

It support following
  • keyboard_arrow_right Drug Brand Name ( Generic drugs brand name list )
  • keyboard_arrow_right Generic Name
  • keyboard_arrow_right Drug – Drug Interaction Search.
  • keyboard_arrow_right Pharmaceutical company Name
  • keyboard_arrow_right Generic dose
  • keyboard_arrow_right Combination of generic name
  • keyboard_arrow_right Search all brand which are 100% similar to other brand
  • keyboard_arrow_right Word Auto completion when you type
  • keyboard_arrow_right Select Brand or Generic from drop down to search respective content.


Following type of Input is supported

  • keyboard_arrow_right Full brand or generic name or pharmaceutical company like Dolo Relieve or Dolo , paracetamol, Medico Health Care etc
  • keyboard_arrow_right combination of any of above like paracetamol Medico Health Care , Dolo paracetamol etc.
  • keyboard_arrow_right Partial Name if you know only part of name of drug or generic name ( minimum of 3 character you should know e.g. para, ceta, medi, dolo ) . Your search query will be like this para* , *ceta* , *mol, i.e. you have to replace the missing characters with * ( without any space )

Input for Drug – Drug Interaction Search

  • keyboard_arrow_right Right Name of any no. of generic drugs separated by space between which interaction you want to search. It also support partial names as mentioned above e.g. paracetamol Probenecid can be searched as para* pro* , *mol pro*, *mol *cid etc. i.e. you have to replace the missing characters with * ( without any space )
